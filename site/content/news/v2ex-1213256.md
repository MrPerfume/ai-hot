它做什么
一句话：网文作者上传小说章节 → 自动生成 3-5 分钟的日系动漫风格短视频。

具体 pipeline：
- 豆包 LLM 解析文本 → 提取角色、场景、对话、自动分镜
- Seedream 按分镜生成动漫画面（参考图保持角色一致性）
- Seedance 2.0 图生视频 + 环境音效
- 火山引擎 TTS 多角色配音
- FFmpeg 合成字幕 → 导出 1080p MP4
整套跑下来一集成本约 ¥9-19（用户自备火山方舟 API Key ），不需要 GPU ，纯编排代码。
