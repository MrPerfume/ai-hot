Arm 联合通义实验室发起「手机上的创意 AI 挑战赛」，赛事时间为 2026/05/19—08/15。

本次挑战赛面向手机端创意 AI 应用开发。

在手机上开发一款创意 AI 应用，至少选用一款 Qwen 系列模型（云端 API 可协同），推荐在 MNN 框架下针对 Arm SME2 指令集进行推理加速与性能调优，让手机算力真正释放。

赛事核心亮点
本次赛事设置 ¥300,000 奖池，入围即享现金或等值实物 + 官方证书。

赛事将设置线下答辩环节，入围团队将在杭州云谷中心现场展示。

为保障公平竞技，晋升团队将统一配发专用手机。

推荐技术栈
模型推荐：Qwen3.5-0.8B/2B/4B、Qwen3-VL-2B/4B，支持结合云端 API（如 Qwen3.6-Plus / Qwen3.5-Omni-Flash）构建“云+端”混合架构，但核心交互逻辑需支持本地运行；
推荐推理框架：MNN（深度适配 Arm SME2 指令集）；
推荐工具：Qwen Code
