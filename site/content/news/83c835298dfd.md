今天，AI 办公领域头部企业像素绽放 PixelBloom 宣布完成 C 轮融资。

本轮融资由国科投资与商汤国香资本联合领投，基石创投、大米创投跟投。

据悉，资金将重点投入 AI 办公解决方案 Agent 的研发迭代、商业化落地及全球化人才招募。

此后，像素绽放 PixelBloom 将围绕三个方向加速布局：
Agent 深度场景化：针对金融、教育、消费、政企四大垂直行业，推出行业专属版本 Agent，进一步贴近实际业务流。

全球化市场拓展：正式进军欧美企业级市场，与国际产品正面竞争。

开发者生态构建：开放 Agent 编排能力与视觉资产 API，吸引第三方开发者基于像素绽放 PixelBloom 平台创建更多垂直场景的 AI 数字员工。

自此，像素绽放 PixelBloom 将从领先的 AI 视觉表达平台，正式向“AI 办公解决方案 Agent”全面跃迁，开启从“生成内容”到“交付结果”的质变。

C 端验证心智，B 端深度嵌合：构筑稳固基本盘
过去两年，像素绽放 PixelBloom 以 PPT 这一高频场景为精准切入点，在全球范围内积累了超过 3000 万注册用户，并在 C 端实现了健康的营收转化。

这一成绩不仅验证了 AI 视觉表达的巨大市场需求，更在用户心智中建立了“AI 做 PPT，就用 AiPPT.com”的品牌认知。

但在更深的水下，像素绽放 PixelBloom 的底牌在于 B 端和政企市场的“无感嵌入”。

公司构建了一套独特的“逆向破局”路径：以 C 端海量用户的真实反馈打磨产品体验，再将成熟稳定的 AI 能力以极低成本、极简集成的方式，渗透至大型组织的工作流深处。

目前，公司的底层生成 API 已成功接入联想智能体“小天”、华为智能体“小艺”以及荣耀智能体“YOYO”的硬件生态，成为这些亿级终端设备中 AI 办公能力的核心供给方。

与此同时，在中信证券、安利中国、农夫山泉等大型企业以及北京、上海、深圳、杭州等一线城市政府机关内部，像素绽放 PixelBloom 的系统已与组织专属知识库深度融合，支撑着数百万名工作人员的日常调用。

这种“C 端验证心智，B 端深度嵌合”的战略，构成了公司现阶段稳固的基本盘，也是其区别于纯 SaaS 厂商或纯 AI 模型公司的核心壁垒。
